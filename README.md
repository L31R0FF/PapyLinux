# PapyLinux
An old computer (**Commodore 64** or **Apple II**) like interface for Linux
# /!\ Warning
You need to be in the sudo group before installation

# Install
`$ unzip papylinux.zip`

`$ cd papylinux`

`$ chmod +x install.sh`

`$ ./install.sh`

# Note : 
you will need to logout and relogin to apply changes 

# Enjoy !!!
